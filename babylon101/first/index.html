First Steps - 第一步

Babylon.JS is a great way to code a 3D environment on the web using the HTML5 canvas element.

Babylon.JS 是一个在 web 上通过 HTML5 canvas 进行 3D 环境编程的极好方式。

The Playground - Playground

This is the quickest and easiest way to make your own scene. Creating a 3D scene is easy, simply add a camera, lights and 3D shapes (meshes) and you are away.

这是创建你自己的场景的最快和最容易的方式(指Playground)。 创建一个三维场景是很容易的，只需要添加一个相机，光源以及三维形状（或是说网格）就搞定了。

The Playground is a web site which has everything you need to create your own scene or edit an existing one. More on the Playground.

[Playground]是一个网站， 它拥有一切你需要用于创建或者编辑自己的场景用到的东西。通过[此处] 了解更多。

A template for creating a scene within the playground is:

在 Playground 中创建场景的模板如下： 

var createScene = function () {

    // Create the scene space
    var scene = new BABYLON.Scene(engine);

    // Add a camera to the scene and attach it to the canvas
    var camera = new BABYLON.ArcRotateCamera("Camera", Math.PI / 2, Math.PI / 2, 2, BABYLON.Vector3.Zero(), scene);
    camera.attachControl(canvas, true);

    // Add lights to the scene
    var light1 = new BABYLON.HemisphericLight("light1", new BABYLON.Vector3(1, 1, 0), scene);
    var light2 = new BABYLON.PointLight("light2", new BABYLON.Vector3(0, 1, -1), scene);

    // This is where you create and manipulate meshes
    var sphere = BABYLON.MeshBuilder.CreateSphere("sphere", {}, scene);

    return scene;

};

Everything else you need is taken care of within the Playground.

你需要做的一切只是在 Playground 里面看清楚就行。

Your Own HTML - 你自己的HTML

When writing your own HTML you just need to embed the createScene function into an HTML page structure with a < script > tag along with a few other items. You will need to load the BabylonJS JavaScript code. Also for use on tablets and mobiles BabylonJS uses pointer events rather than mouse events and so the PEP event system needs to be loaded as well.

当你要写自己的HTML时你只需要将 createScene 函数用一个 <script>标签嵌入到HTML页面中。 另外需要附带一些少量的其他东西，你需要加载 BabylonJS 的 JavaScript 代码， 另外，如果使用平板或者手机的话，BabylonJS 使用指针事件而不是鼠标事件，因此 PEP 事件系统也必须要加载。

In addition a canvas element will have to be added to the body as this is where the 3D scene will be rendered and a reference variable canvas added to it in the code. You also need to generate the BabylonJS engine before the function for creating the scene.

另外在 body 中必须要加入一个 canvas 元素，因为这就是三维场景用于渲染的地方，代码中也要将 canvas 作为变量引入。 你也需要在函数创建场景之前引入 BabylonJS 引擎。

Finally, add code to call the scene. This enables the engine to continually render the scene in a loop and to resize it if the browser is ever resized.

最后，添加代码来调用场景。这使得引擎可以持续对场景进行持续的循环渲染，并且随着浏览器窗口大小变化，让场景大小随之变化。

HTML template - HTML模板

<!DOCTYPE html>
<html xmlns="http://www.w3.org/1999/xhtml">

    <head>
        <meta http-equiv="Content-Type" content="text/html; charset=utf-8"/>
        <title>Babylon Template</title>

        <style>
            html, body {
                overflow: hidden;
                width: 100%;
                height: 100%;
                margin: 0;
                padding: 0;
            }

            #renderCanvas {
                width: 100%;
                height: 100%;
                touch-action: none;
            }
        </style>

        <script src="https://cdn.babylonjs.com/babylon.js"></script>
        <script src="https://preview.babylonjs.com/loaders/babylonjs.loaders.min.js"></script>
        <script src="https://code.jquery.com/pep/0.4.3/pep.js"></script>
    </head>

   <body>

    <canvas id="renderCanvas" touch-action="none"></canvas> //touch-action="none" for best results from PEP

    <script>
        var canvas = document.getElementById("renderCanvas"); // Get the canvas element
        var engine = new BABYLON.Engine(canvas, true); // Generate the BABYLON 3D engine

        /******* Add the create scene function ******/
        var createScene = function () {

            // Create the scene space
            var scene = new BABYLON.Scene(engine);

            // Add a camera to the scene and attach it to the canvas
            var camera = new BABYLON.ArcRotateCamera("Camera", Math.PI / 2, Math.PI / 2, 2, new BABYLON.Vector3(0,0,5), scene);
            camera.attachControl(canvas, true);

            // Add lights to the scene
            var light1 = new BABYLON.HemisphericLight("light1", new BABYLON.Vector3(1, 1, 0), scene);
            var light2 = new BABYLON.PointLight("light2", new BABYLON.Vector3(0, 1, -1), scene);

            // Add and manipulate meshes in the scene
            var sphere = BABYLON.MeshBuilder.CreateSphere("sphere", {diameter:2}, scene);

            return scene;
        };
        /******* End of the create scene function ******/

        var scene = createScene(); //Call the createScene function

        // Register a render loop to repeatedly render the scene
        engine.runRenderLoop(function () {
                scene.render();
        });

        // Watch for browser/canvas resize events
        window.addEventListener("resize", function () {
                engine.resize();
        });
    </script>

   </body>

</html>

Notes - 注释

  1. The examples above use the newer MeshBuilder method for creating shapes where variables for the shape are set within the options object parameter and has some advantages over the older form of BABYLON.Mesh.Create.... which uses a parameter list for the shape variables. The majority of Playgrounds use the older method as many were created before MeshBuilder existed.

    上面的例子用了新版的 MeshBuilder 方法来创建形状，形状的变量是在 options 对象参数里面设置的， 这样相比起旧版的方式（BABYLON.Mesh.Create，并使用参数列表传入形状的变量）有一些优势。 多数的 Playground 样例还是会用回旧的形式，因为很多的例子是在 MeshBuilder 问世之前被创建的。

  2. The use of PEP for pointer events is more recent advice, older advice was to use a system called hand.js. Both work, although hand.js is no longer maintained. You may still find references to hand.js in the documentation.

    使用 PEP 来处理指针事件是一个最近的建议，旧的建议是使用一个叫做 hand.js 的系统。 它们都可以工作，尽管 hand.js 已经停止维护了。你仍然可以在文档中找到 hand.js 的引用。
