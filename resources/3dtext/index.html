Make 3D Text

Produced by forum user vinhpt this 3D Text site, together with other apps, acknowledged on site by the author, allows you to make and export Babylon.js ready 3D text like this:

Default

Cutout using Throughout option

Outline using Stroke option

Use is very straight forward; enter your text, decide on thickness (depth) and styles and then hit the Vectorize → 3D button followed by the Download button.
